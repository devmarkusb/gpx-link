## Desktop Ad Monetization

### Start Here

- [ ] Apply to [Amazon Associates](https://affiliate-program.amazon.com/) for GPS units, hiking gear, power banks, bike accessories, and related outdoor products.
- [ ] Apply to the [Booking.com Affiliate Program](https://www.booking.com/affiliate-program/v2/index.html) for travel and accommodation offers. Booking.com provides links, banners, and affiliate tools through CJ.
- [ ] Apply to [Awin publishers](https://www.awin.com/us/publishers) for brands such as Decathlon, Booking.com, Sixt, TUI, Viator, and other outdoor/travel partners.
- [ ] Apply to [CJ Publisher](https://www.cj.com/publisher) for larger brand affiliate programs and travel/outdoor partners.

### Workflow

1. [ ] Sign up as a publisher or affiliate.
2. [ ] Search for outdoor, travel, and GPS brands.
3. [ ] Get approved by the brand.
4. [ ] Copy the approved tracking link or banner creative.
5. [ ] Put it into a hosted `banner.html`.
6. [ ] Point the desktop app at it:

```bash
GPX_LINK_DESKTOP_AD_URL="https://your-domain.com/gpx-link/banner.html" gpx-link-gui
```
